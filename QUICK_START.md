# Curt 减脂计算器 - 快速开始指南

## 📱 微信开发者工具预览步骤

### 第一步：下载微信开发者工具

1. 访问 [微信开发者工具官网](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 选择对应操作系统版本下载（Windows/Mac）
3. 安装完成后打开应用

### 第二步：导入项目

1. 打开微信开发者工具
2. 点击左上角"导入项目"按钮
3. 在弹出的对话框中：
   - **项目文件夹**: 选择本项目的根目录 (`curt-fat-loss-calculator`)
   - **AppID**: 输入 `wx1234567890abcdef`（测试号）或使用自己的AppID
   - **项目名称**: 输入 `Curt 减脂计算器`
   - **开发框架**: 保持默认
4. 点击"导入"按钮

### 第三步：预览小程序

1. 项目导入成功后，点击顶部工具栏的"预览"按钮
2. 微信开发者工具会生成一个二维码
3. 用微信扫描二维码
4. 在微信中即可看到小程序的实时预览

### 第四步：调试

- **Console 控制台**: 查看 JavaScript 错误和日志
- **Network 网络**: 查看网络请求（本项目无后端请求）
- **Storage 存储**: 查看本地存储数据
- **Sources 源代码**: 设置断点调试代码

## 🎯 功能使用流程

### 1️⃣ 热量计算（第一个Tab）

**目的**: 计算每日热量消耗和营养素分配

**操作步骤**:
1. 输入基本信息
   - 选择性别：男性 / 女性
   - 输入年龄：15-100岁（默认22岁）
   - 输入身高：100-250cm（默认172cm）
   - 输入体重：30-200kg（默认60kg）

2. 系统自动计算基础代谢率(BMR)
   - 男性: 10×体重 + 6.25×身高 - 5×年龄 + 5
   - 女性: 10×体重 + 6.25×身高 - 5×年龄 - 161

3. 输入训练信息
   - 每日训练时间：0-300分钟（默认0）
   - 选择训练强度：
     - 新手（系数5）
     - 健身爱好者（系数8，默认）
     - 健身老手（系数10）

4. 选择营养素比例
   - **532比例**: 碳水50% / 蛋白质30% / 脂肪20%
   - **442比例**: 碳水40% / 蛋白质40% / 脂肪20%
   - **自定义**: 手动输入三个百分比（总和必须为100%）

5. 查看结果
   - 每日总热量消耗（TDEE）
   - 碳水、蛋白质、脂肪的克数和热量

**示例数据**:
- 输入：22岁男性，172cm，60kg，训练时间0分钟，爱好者强度
- 结果：BMR约1570千卡，TDEE约1570千卡
- 营养素（532比例）：碳水196克、蛋白质117克、脂肪34克

### 2️⃣ 减脂预测（第二个Tab）

**目的**: 预测减脂周期和减重计划

**操作步骤**:
1. 输入初始体重：当前体重（默认60kg）
2. 输入目标体重：想要达到的体重（默认50kg）
3. 调整每月减重比例：
   - 点击"−"减少，点击"+"增加
   - 范围：1% - 5%（默认3%）
   - 建议：3%左右较为健康

4. 查看预测结果
   - 总共需要减重的公斤数
   - 预计需要的月数（以及周数和天数）
   - 每月需要减重的公斤数
   - 每周需要减重的公斤数

5. 点击"保存目标"保存数据到本地

**示例数据**:
- 初始体重：60kg，目标体重：50kg，每月减重比例：3%
- 结果：总减重10kg，预计3.33月（14周3天）

### 3️⃣ 个人信息（第三个Tab）

**目的**: 查看身体数据和BMI评估

**显示内容**:
1. **个人信息卡片**
   - 每日总热量建议
   - 年龄、体重、身高

2. **营养素分配建议**
   - 碳水、蛋白质、脂肪的克数和热量
   - 进度条展示比例

3. **体重变化**
   - 目标体重 vs 当前体重
   - 减重进度百分比

4. **BMI指数**
   - BMI数值和状态判断
   - 彩色进度条展示范围
   - BMI分类说明

5. **常见问题**
   - 点击问题可展开/收起答案
   - 包含关于小程序的常见问题

## 📊 数据计算示例

### 热量计算示例
```
输入：
- 性别：男性
- 年龄：25岁
- 身高：175cm
- 体重：70kg
- 训练时间：60分钟
- 训练强度：爱好者（系数8）
- 营养素比例：532

计算过程：
1. BMR = 10×70 + 6.25×175 - 5×25 + 5 = 700 + 1093.75 - 125 + 5 = 1673.75 ≈ 1674千卡
2. 训练热量 = 60 × 8 = 480千卡
3. TDEE = 1674 + 480 = 2154千卡
4. 碳水 = (2154 × 50%) / 4 = 269克
5. 蛋白质 = (2154 × 30%) / 4 = 161克
6. 脂肪 = (2154 × 20%) / 9 = 48克
```

### 减脂预测示例
```
输入：
- 初始体重：70kg
- 目标体重：65kg
- 每月减重比例：2.5%

计算过程：
1. 总减重 = 70 - 65 = 5kg
2. 每月减重 = 70 × 2.5% = 1.75kg
3. 预计月数 = 5 / 1.75 = 2.86月
4. 预计天数 = 2.86 × 30 = 85.8天
5. 预计周数 = 85.8 / 7 = 12周2天
6. 每周减重 = 1.75 / 4 = 0.44kg
```

## ⚠️ 注意事项

1. **数据准确性**
   - 输入的身体数据越准确，计算结果越准确
   - 建议定期更新体重数据

2. **健康建议**
   - 每月减重比例建议在2%-3%
   - 过快减重可能影响健康
   - 建议咨询专业营养师

3. **浏览器兼容性**
   - 仅支持微信客户端
   - 需要微信 iOS 7.0+ 或 Android 7.0+

4. **数据存储**
   - 本地存储的数据仅在本设备保留
   - 卸载小程序会清除本地数据
   - 建议定期记录重要数据
